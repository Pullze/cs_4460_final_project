## How to Run

### Requirements
- Node.js @ 16.0 or higher
- `npm`

### Install Dependencies

In the project directory, run:

#### `npm install`

### Run the App

#### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
